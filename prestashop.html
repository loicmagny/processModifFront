Installation d'un PrestaShop existant en local

  1. Tout d'abord, vous aurez besoin d'un environnement de développement local comme MAMP (pour Mac), WAMP (pour Windows), ou LAMP (pour Linux). Téléchargez et installez le logiciel correspondant à votre système d'exploitation.
  2. Démarrez le serveur local et notez l'adresse du serveur (généralement localhost) et le numéro de port.
  3. Créez une nouvelle base de données via l'interface de PhpMyAdmin (ou équivalent) fournie avec votre serveur local. Notez le nom de la base de données, l'utilisateur et le mot de passe.
  4. Importez la sauvegarde de la base de données de votre site PrestaShop dans la base de données que vous venez de créer. Vous pouvez le faire en utilisant l'option Import dans PhpMyAdmin.
  5. Extrayez la sauvegarde des fichiers de votre site PrestaShop dans le dossier htdocs (ou www pour WAMP) de votre serveur local. Assurez-vous que les fichiers sont dans un sous-dossier, par exemple localhost:8888/my-prestashop.
  6. Modifiez le fichier settings.inc.php dans le dossier /config de votre PrestaShop local. Changez les valeurs de _DB_NAME_, _DB_USER_, _DB_PASSWD_ pour correspondre à votre base de données locale.
  7. Ouvrez votre navigateur et accédez à votre site PrestaShop via l'URL de votre serveur local, par exemple http://localhost:8888/my-prestashop.
  8. Si vous rencontrez des problèmes d'URL (par exemple, le site essaie de rediriger vers l'URL en direct), vérifiez la table ps_configuration dans votre base de données et recherchez les valeurs PS_SHOP_DOMAIN et PS_SHOP_DOMAIN_SSL. Assurez-vous qu'ils sont définis sur votre adresse de serveur local.
