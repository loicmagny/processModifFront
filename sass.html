Tutoriel rapide sur SCSS

  1. Le SCSS (Sassy CSS) est une extension de CSS qui ajoute de nouvelles fonctionnalités et améliorations aux capacités existantes de CSS.
  2. Vous pouvez déclarer des variables en SCSS avec le signe $. Par exemple :

    $primary-color: #ff6347;

    Ces variables peuvent ensuite être utilisées dans votre code SCSS.
  3. SCSS supporte les imbrications, ce qui signifie que vous pouvez définir des styles pour des éléments enfants directement à l'intérieur des règles de leurs parents. Par exemple :

    nav {
      background-color: $primary-color;
      a {
        color: white;
      }
    }

  4. SCSS offre le support des opérateurs mathématiques. Vous pouvez utiliser des opérations arithmétiques comme l'addition, la soustraction, la multiplication et la division dans vos styles. Par exemple :

    .container {
      width: 100%/3;
    }

  5. Avec SCSS, vous pouvez créer des mixins, qui sont des blocs de code réutilisables. Vous pouvez les créer avec @mixin et les utiliser avec @include. Par exemple :

    @mixin reset-list {
      margin: 0;
      padding: 0;
      list-style: none;
    }

    ul {
      @include reset-list;
    }

  6. Les mixins peuvent également prendre des arguments, ce qui les rend encore plus flexibles. Par exemple :

    @mixin border-radius($radius) {
      border-radius: $radius;
    }

    .box {
      @include border-radius(10px);
    }

  7. SCSS fournit aussi les boucles, comme @for, @each, @while, qui sont très utiles pour générer des styles répétitifs. Par exemple :

    @for $i from 1 through 3 {
      .col-#{$i} { width: 100%/$i; }
    }

  8. Le SCSS permet également l'utilisation de fonctions. Par exemple, vous pouvez utiliser la fonction lighten pour éclaircir une couleur :

    nav {
      background-color: lighten($primary-color, 10%);
    }

  9. SCSS comprend un système de gestion des erreurs avec @warn et @error. Ces directives permettent d'afficher des avertissements ou de générer des erreurs lors de la compilation du SCSS.

    @mixin assert-integer($value) {
      if (type-of($value) != integer) {
        @error "Expected an integer value, but got #{type-of($value)}.";
      }
    }

Pour plus d'informations et de détails, vous pouvez consulter la documentation officielle de SCSS.
