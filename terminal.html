Windows CMD

  1. Utilisez la commande cd pour naviguer entre les dossiers.

    Exemple : cd Documents

  2. La commande dir affiche le contenu du dossier courant.
  3. La commande cls permet de nettoyer le terminal.
  4. La commande taskkill permet de forcer l'arrêt d'un processus.

    Exemple : taskkill /IM notepad.exe

  5. Utilisez Ctrl + C pour forcer l'arrêt d'un processus en cours dans le terminal.

macOS et Linux (Ubuntu / Debian) Terminal

  1. Utilisez la commande cd pour naviguer entre les dossiers.

    Exemple : cd Documents

  2. La commande ls affiche le contenu du dossier courant.
  3. La commande ls -la affiche le contenu du dossier courant, y compris les fichiers cachés (les fichiers commençant par un point, par exemple .htaccess).
  4. La commande clear permet de nettoyer le terminal.
  5. La commande kill permet de forcer l'arrêt d'un processus.

    Exemple : kill 12345 (où 12345 est l'ID du processus)

  6. Utilisez Ctrl + C pour forcer l'arrêt d'un processus en cours dans le terminal.
